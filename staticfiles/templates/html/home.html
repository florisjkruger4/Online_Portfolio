Storybook Generator

   

Using OpenAI's Chat GTP-3 API to generate a kids storybook with an artificially drawn cover image and text completion. This application allows the user to select from four different storybook themes and answer three short story prompts within the selected theme to generate the desired storybook. Python's gTTS (Google Text-to-Speech) library is also used to interface with Google Translate test-to-speech API in order to read the generated storybook out loud back to the user if they desire to do so.

The different storybook themes are Fantasy, Sports, Adventure and Spooky. The following is an example of a fantasy themed storybook that was generated by answering the following prompts in the multistep form.

  • Enter a name! - Scooby Doo
  • Enter your story's quest! - rescueing a princess
  • Enter a location! - a medieval kingdom
Hints for the prompt are also providded in case the user gets stuck or cannot think of anything. New suggestions are provided upon refreshing the page.

 

Yellowmap

   

An interactive website which incorporates geospatial data on Yellowstone National Park and the use of the Mapbox API. This application provides a map view of Yellowstone with togglable filters to give more information about the park. Yellowmap also provides directional functionality in order to get to a destination, along with written G.P.S. instructions and gives an estimated time of arrival by car.

Users are also able to upload images and descriptions of various wildlife animals that are found in the park and set a marker with specific latitude and longitude coordinates of where the animal was observed. The posts made by users are logged into the Django admin databse and can be viewd on the posts page along with their descriptions, pictures and timestamps.

 

Sorting Visualizer

 

Programmed in C++, this program makes use of the software development open source library called Raylib which is used to design graphical applications. In this program, Raylib is used to help visually show various sorting algorithms and how they function in order to sort a set number of quantities in an array which are reperesented by rectangular bars that have different heights according to their value.

This sorting visualizer showcases four different sorting algorithms including bubble sort, selection sort, insertion sort and quick sort. The program also provides a randomize array option which generates a completely new and randomly sorted selection of bars within the array.

 

Clear The Forest

 

This 2D platformer game is programmed in C# within the Unity Game Engine. All the art was designed completely from scratch using Adobe Photoshop and includes a wide variety of different pixel art sprites. The game involves navigating around a forest and using two different weapon types to kill various monsters and collect gems. All assets were designed uniquely from scratch.

The game focuses on giving player feedback to interact with various items throughout the level and explore which weapon type works best in different situations. The opjective of the game is quite simple but it is visually pleasing and has responsive gameplay.
